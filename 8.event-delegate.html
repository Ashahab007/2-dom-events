Event Delegate

(Event Delegation) হলো JavaScript এর একটি কৌশল যা DOM ইভেন্টগুলিকে প্যারেন্ট উপাদান থেকে পরিচালনা করতে সাহায্য করে, যাতে আপনি অনেকগুলো চাইল্ড উপাদানের জন্য একক ইভেন্ট হ্যান্ডলার ব্যবহার করতে পারেন। এর মাধ্যমে, আপনি একাধিক উপাদানে আলাদা আলাদা ইভেন্ট হ্যান্ডলার যোগ করার পরিবর্তে প্যারেন্ট উপাদানে একটিমাত্র ইভেন্ট হ্যান্ডলার ব্যবহার করতে পারেন। এটি ডাইনামিক উপাদানগুলির জন্য খুবই কার্যকর, কারণ নতুন উপাদান যুক্ত হলেও কোনো সমস্যা হয় না।
