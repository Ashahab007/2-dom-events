Concepts of Event Bubble

Event Bubble ৩ টা stage এ কাজ করে। See the flow chart|

  • Capture stage
  • Target Stage
  • Bubble stage

Theoritically প্রথমে capturing stage এ টার্গেট element টা খুজে বের করে যেখানে event টা trigger হয়। এরপর টার্গেট element টা capture করে bubbling এর মত উপরে উঠে result show করে। কিন্তু practically capture টা হয় হচ্ছে bubbling stage এ। যার কারনে bubble stage এ যদি অন্য sectoin এ event handler set করা হয় সেটা বোঝা যায়। see the following example with console. এখানে সবগুলো stage এ event handler set করা আছে। যদি item 2 তে click করা হয় তখন target element capture হয় সাথে set করা অন্য section এর event handler এর message গুলো bubbling এর সময় capture হয়ে sequentially result এ output দেখায়।

stopPropagation: stopPropagation এর মাধ্যমে function এ event parameter নিয়ে event bubbling stop করা যায়। To see the result commented out the event.stopPropagation()
